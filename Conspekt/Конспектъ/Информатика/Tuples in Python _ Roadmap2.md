В Python существует несколько встроенных типов данных для хранения коллекций элементов. Одним из самых важных и часто используемых является **tuple** — кортеж. Кортеж представляет собой упорядоченную и неизменяемую коллекцию объектов. Это означает, что элементы внутри tuple располагаются в определённом порядке и после создания не могут быть изменены.

Кортежи очень похожи на списки (`list`), однако между ними есть важное различие: список можно изменять, добавлять и удалять элементы, а tuple остаётся неизменным на протяжении всего времени существования объекта. Благодаря этому кортежи работают быстрее списков и используются в ситуациях, когда данные должны оставаться постоянными.

Tuple активно применяется в реальных программах: для хранения координат, параметров настройки, записей баз данных, результатов функций и других структур, которые не предполагают изменения.
# Создание Tuple

Создать кортеж в Python можно несколькими способами. Чаще всего используются круглые скобки:

```
numbers = (1, 2, 3, 4, 5)
```

Кортеж может содержать любые типы данных:

```
data = ("Alice", 25, 175.5, True)
```

Python позволяет создавать tuple даже без круглых скобок:

```
person = "Bob", 30
```

В этом случае Python автоматически понимает, что создаётся кортеж.
# Tuple из одного элемента

Одной из особенностей tuple является создание кортежа с одним элементом. Многие начинающие программисты совершают ошибку, забывая поставить запятую.

Правильный вариант:

```
single = (10,)
```

Неправильный вариант:

```
single = (10)
```

Во втором случае Python создаст не tuple, а обычное число типа `int`.

Запятая является обязательным признаком кортежа из одного элемента.

# Доступ к элементам

Tuple поддерживает индексацию так же, как и списки. Первый элемент имеет индекс `0`.

```
colors = ("red", "green", "blue")print(colors[0])print(colors[1])
```

Можно использовать отрицательные индексы:

```
print(colors[-1])
```

Отрицательная индексация позволяет обращаться к элементам с конца коллекции.
