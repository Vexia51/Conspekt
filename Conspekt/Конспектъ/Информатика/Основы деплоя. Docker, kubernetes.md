Docker - инструмент для контейнеризации, который упаковывает приложение с зависимостями в лёгкий изолированный контейнер. Он изолирует процессы на уровне ядра и работает на одном хосте. Его задача увеличивать совместимость, упростить деплой и воспроизводимость.

Деплой - процесс развёртывания приложения на хостинге, для обеспечения внешнего доступа.

Kubernets - инструмент для управления контейнерами, обеспечивающий высокую эффективность и отказоустойчивость развёрнутых систем.
