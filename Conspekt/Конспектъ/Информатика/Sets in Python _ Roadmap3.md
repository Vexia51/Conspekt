#Roadmap
Set в Python - это неупорядоченная коллекция уникальных элементов. Она мгновенно проверяет наличие объекта (за O(1)) благодаря хэш-таблицам. Дубликаты внутри удаляются автоматически.
Элементами могут быть только неизменяемые типы: числа, строки, кортежи. Пустое множество создается как `set()`, а не `{}` (это словарь). Для изменения состава используют `add()`, `update()`, `remove()` и безопасный `discard()`. Множества поддерживают математические операции: объединение `|`, пересечение `&`, разность `-` и симметрическую разность `^`. Главные сценарии применения: быстрое удаление дубликатов из списков и моментальный поиск через `in`
