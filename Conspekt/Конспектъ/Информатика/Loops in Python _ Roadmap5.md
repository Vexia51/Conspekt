#Roadmap
Циклы в Python - это конструкции для автоматического повторения кода, которые делятся на два типа: `for` и `while`.
- Цикл `for` используется для перебора последовательностей, таких как списки или диапазоны чисел, и удобен при работе с коллекциями данных.
- Цикл `while` продолжает выполнение до тех пор, пока заданное условие остается истинным, что полезно для задач с неопределенным числом итераций, например, при обработке пользовательского ввода.

Управление ходом работы циклов осуществляется операторами `break` для принудительного выхода и `continue` для пропуска текущего шага, а блок `else` позволяет выполнить код после корректного завершения итераций
