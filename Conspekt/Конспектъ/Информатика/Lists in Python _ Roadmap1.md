Списки — это фундаментальные структуры данных в Python, используемые для хранения упорядоченной коллекции элементов. Эти элементы могут быть разных типов данных (числа, строки или даже другие списки), а списки могут быть изменчивыми, то есть их содержимое можно изменять после создания путём добавления, удаления или изменения элементов. Они определяются с помощью квадратных скобок, а элементы разделены запятыми.`[]`

## Что такое список

Список — это изменяемая коллекция элементов. В списке можно хранить числа, строки, логические значения и даже другие списки.

Список записывается в квадратных скобках:

```
colors = ["blue", "green", "red", "yellow"]
```

# Создание списков

## Список строк

```
names = ["Olivia", "Nathan", "Bethany", "Jacob"]
```

## Список чисел

```
numbers = [1, 2, 3, 4, 5]
```

## Смешанный список

```
mixed = [1, "Python", True, 3.14]
```

## Пустой список

```
empty = []
```

# Доступ к элементам списка

Элементы списка имеют индексы. Нумерация начинается с `0`.

```
colors = ["blue", "green", "red"]print(colors[0])
```

Результат:

```
blue
```

Можно использовать отрицательные индексы:

```
print(colors[-1])
```

Результат:

```
red
```

# Изменение элементов списка

Главная особенность списка — его можно изменять.

```
numbers = [1, 2, 3]numbers[1] = 10print(numbers)
```

Результат:

```
[1, 10, 3]
```

# Добавление элементов

## append()

Добавляет элемент в конец списка.

```
numbers = [1, 2]numbers.append(3)
```

## insert()

Вставляет элемент по индексу.

```
numbers.insert(1, 10)
```

# Удаление элементов

## remove()

Удаляет элемент по значению.

```
numbers.remove(10)
```

## pop()

Удаляет последний элемент.

```
numbers.pop()
```

## del

Удаляет элемент по индексу.

```
del numbers[0]
```

# Срезы списков

Срезы позволяют получать часть списка.

```
nums = [0, 1, 2, 3, 4, 5]print(nums[1:4])
```

Результат:

```
[1, 2, 3]
```

Каждый второй элемент:

```
print(nums[::2])
```

# Перебор списка

## Цикл for

```
colors = ["red", "green", "blue"]for color in colors:    print(color)
```

## enumerate()

Позволяет получить индекс и значение.

```
for i, color in enumerate(colors):    print(i, color)
```

# Проверка наличия элемента

```
fruits = ["apple", "banana"]print("apple" in fruits)
```

# Длина списка

```
numbers = [1, 2, 3]print(len(numbers))
```

# Сортировка списков

```
nums = [3, 1, 5, 2]nums.sort()print(nums)
```

Обратная сортировка:

```
nums.sort(reverse=True)
```
