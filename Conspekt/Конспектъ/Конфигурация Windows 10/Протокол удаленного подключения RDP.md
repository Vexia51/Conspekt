#Windows #RDP #ПК 
RDP (Remote Desktop Protocol) - проприетарный протокол Microsoft для удаленного управления рабочим столом Windows (порт 3389). Он передает графический интерфейс на локальный компьютер, а сигналы мыши и клавиатуры — на удаленный, обеспечивая безопасность с помощью шифрования и NLA.

Для подключения используется встроенная утилита **mstsc**. Чтобы ее запустить, нажмите Win + R, введите mstsc и нажмите Enter. В открывшемся окне укажите IP-адрес или имя удаленного компьютера. При необходимости можно развернуть «Показать параметры» для настройки экрана, проброса локальных устройств (принтеров, буфера обмена) и оптимизации трафика.

Нажмите Подключить, подтвердите сертификат безопасности (если появится предупреждение) и введите логин с паролем от учетной записи удаленного ПК. После этого откроется рабочий стол удаленной системы.

Чтобы свернуть сеанс с сохранением всех работающих программ в фоне, закройте окно за крестик на верхней панели. Чтобы полностью завершить сеанс и закрыть приложения, выполните выход (Sign out) через меню «Пуск» внутри удаленной системы.
