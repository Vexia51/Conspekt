# OSPanel
OSPanel (Open Server Panel) — портативный локальный веб-сервер для Windows, объединяющий все необходимые инструменты для веб-разработки в одном архиве.

## История создания

Появился в 2010-х как ответ на неудобства существующих решений вроде XAMPP или WampServer: сложная установка, конфликты портов, нестабильность.  
Разработчик (Андрей Стрельников) сделал акцент на простоте, портативности и поддержке множества PHP-версий — сейчас это v6+ с миллионами скачиваний.

## Ключевые задачи

- **Управление стеками**: Apache/Nginx + PHP (5.3–8.3, 115+ расширений), MySQL 5.7–8.4, MariaDB, PostgreSQL, Redis, Memcached, MongoDB — переключение версий без переустановки.
- **Локальная разработка**: Виртуальные хосты (*.localhost), SSL, IPv6, FTP/SFTP, cron, Composer, Node.js, Python, Mailpit для email-тестирования.
- **Автоматизация**: Мониторинг процессов, логи, резервное копирование БД, oha для нагрузки, поддержка Docker-контейнеров.

## Системные требования и установка

**Требования**: Windows 7–11 (x86/x64), 512MB+ RAM, ~1GB места.  
**Установка**: Скачать .7z с ospanel.io, распаковать в любую папку (без пробелов/русских букв), запустить `OpenServer.exe` — сервер готов за 1 минуту. 

```
Структура папок:
├── domains/     # Сайты (*.localhost)
├── modules/     # Версии PHP/БД
├── userdata/    # Конфиги, БД
└── userdir/     # Пользовательские файлы
```

## Применение в проектах

| Сценарий | Инструменты OSPanel | Преимущества |
|----------|-------------------|-------------|
| **PHP-сайты (WordPress, Laravel)** | PHP 8.2 + MySQL + Nginx | Быстрый импорт БД, виртуальные хосты |
| **Frontend (Node.js/React)** | Node 20 + npm + Apache | Прокси на localhost:3000 |
| **API/Backend** | PHP-FPM + PostgreSQL + Redis | Мониторинг, логи в реальном времени |
| **Командная работа** | Локальная сеть (192.168.x.x) | Доступ по IP, SSL для тестов |

**Примеры использования**: Фрилансеры — быстрая настройка под клиента; студенты — изучение PHP/SQL; команды — синхронизация через Git + общий доступ.
