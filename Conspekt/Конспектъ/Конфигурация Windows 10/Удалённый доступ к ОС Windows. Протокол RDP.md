## Введение 
Процесс удаленного подключения к другой системе на ос Windows можно осуществить несколькими различными способами, один из которых является подключение через встроенную утилиту подключения Windows 10, реализующую протокол RDP.

## История RDP 
Протокол RDP был разработан компанией Microsoft для удаленного управления компьютерами.
